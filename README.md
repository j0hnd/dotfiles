# Dotfiles

## Installation

#### Install jellybean for vim
    
    mkdir ~/.vim/colors                                                                                                
    wget https://raw.githubusercontent.com/nanotech/jellybeans.vim/master/colors/jellybeans.vim ~/.vim/colors/
